# ☁️ Cloud Computing Lab – Sales Data Analysis using Google Sheets

This project demonstrates how **Google Sheets can be used as a cloud-based data analysis tool** for performing data cleaning, analysis, and visualization.  

The lab also highlights **SaaS collaboration features**, allowing multiple users to work on the same dataset, add comments, make edits, and track changes in real time.

## Overview

This cloud computing lab focuses on using **Google Sheets as a SaaS platform** to analyze sales-related data. The project involves cleaning the dataset, creating pivot tables, building charts, and designing a simple dashboard to present insights clearly.

It also demonstrates how cloud platforms support **team collaboration and version tracking**.

---

## Project Objective

- Clean and organize raw sales data  
- Perform analysis using **Pivot Tables**  
- Create visual insights with **Charts**  
- Design a **simple analytical dashboard**  
- Experience **real-time collaboration in Google Sheets**

---

## 📂 Dataset

The dataset contains information related to:

- Sales regions  
- Product categories  
- Salespersons  
- Products  
- Revenue and performance data  

This data was used to perform analysis and extract business insights.

## Dashboard Features

The dashboard summarizes important insights from the dataset, including:

- Sales performance by **region**
- Profitability by **product category**
- Best performing **salesperson**
- Most sold **product**
