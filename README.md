# Sales Data Analysis using Google Sheets

This project demonstrates how **Google Sheets can be used as a cloud-based data analysis platform** for performing **data cleaning, analysis, and visualization**.

The lab also highlights **Software as a Service (SaaS) collaboration features**, allowing multiple users to work on the same dataset, add comments, make edits, and track changes in **real time**.

## Overview

This **Cloud Computing Lab project** focuses on using **Google Sheets as SaaS platform** to analyze sales data.

The project demonstrates how cloud-based tools can support:

- **Data cleaning and preparation**
- **Exploratory data analysis**
- **Data visualization**
- **Dashboard creation**
- **Real time collaboration**

Using Google Sheets, the dataset was cleaned, analyzed with **Pivot Tables**, and visualized through **interactive charts** to build a simple **sales performance dashboard**.

---

## Project Objective

The main goals of this lab were:

- Clean and organize **raw sales data**
- Perform analytical operations using **Pivot Tables**
- Create **visual charts for insights**
- Build a **simple analytical dashboard**
- Understand **cloud based SaaS collaboration**
- Track changes using **version history**

## Project Workflow

The project followed a structured **data analysis workflow**.

### Data Collection
- Import the sales dataset into Google Sheets.

### Data Cleaning
- Remove duplicates  
- Handle missing values  
- Standardize column formats  

### Data Analysis
- Create **Pivot Tables**
- Aggregate sales metrics

### Data Visualization
- Build charts for visual insights

### Dashboard Development
- Combine charts and metrics into a **simple dashboard**

## Cloud Collaboration Features
